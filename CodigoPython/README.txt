Introducción:
------------

El código expuesto en Pregunta2_a.py busca mostrar empíticamente la ley de los grandes números haciendo la preuba con 2 distribuciones diferentes. Se usarán las distribuciones gamma y beta.

Instalación:
-----------

Si Usted desea ejecutar el código primero debe instalar python, matlibplot y numpy.

En ubuntu:
---------

	sudo apt-get install python python-numpy python-matlibplot

En windows (No ha sido probado aún en windows):
----------------------------------------------

	Python en windows: 
		Descar desde el siguiente link y luego ejecutarlo
		https://www.python.org/downloads/release/python-350a4/ 

	numpy en windows:
		Seguir las instrucciones dispuestas en el siguiente link.
		http://www.scipy.org/scipylib/download.html

	matpotlib en windows:	
		Seguir las instrucciones dispuestas en el siguiente link.
		http://matplotlib.org/downloads.html

Uso:
---

El uso es simplemente la ejecución del programa, el cual no espera por parametros de entrada.

Pregunta 2.a:
	python Pregunta2_a.py
